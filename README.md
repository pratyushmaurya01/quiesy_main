## 📖 About Quiesy

**Quiesy** is a full-stack, premium assessment platform designed to bridge the gap between traditional quizzes and technical evaluations. Built with a focus on User Experience (UX), Quiesy offers a distraction-free, NTA-style testing environment for students, and a powerful, analytics-driven dashboard for educators.

Whether it's a standard Multiple Choice Question (MCQ) test or a complex Coding Challenge requiring real-time compilation, Quiesy handles it all seamlessly within a beautiful Dark/Light mode supported UI.

---

## ✨ Key Features

### 👨‍🏫 For Educators (The Workspace)
* **Dual Assessment Modes:** Create standard MCQs or advanced Coding problems with starter code.
* **Hidden Test Cases:** Add custom input/output test cases for automated code evaluation.
* **Deep Analytics Dashboard:** Track student performance, view score distributions via interactive charts, and export results directly to CSV.
* **Access Control:** Secure quizzes with 4-digit passcodes and toggle "Student Review" visibility on or off in real-time.
* **Shareable Links:** 1-click clipboard copying for easy quiz distribution.

### 🧑‍💻 For Students (The Arena)
* **Pro Coding Environment:** Integrated **Monaco Editor** (VS Code engine) with syntax highlighting for Python, Java, and C++.
* **Real-Time Evaluation:** Run code against hidden test cases directly in the browser and get instant terminal feedback.
* **Split-Pane Layout:** A LeetCode-style resizable interface that keeps the problem description and code editor side-by-side.
* **Distraction-Free UX:** Auto-saving, intuitive navigation queues, and a clean interface to ensure maximum focus.

## 🛠️ Technology Stack

### **Frontend**
* **Framework:** React.js (Vite)
* **Styling:** Tailwind CSS (Fully responsive + Dark/Light Mode)
* **Routing:** React Router DOM
* **Code Editor:** `@monaco-editor/react`
* **Charts:** Recharts (For analytics visualization)

### **Backend**
* **Framework:** Django & Django REST Framework (DRF)
* **Authentication:** JWT (JSON Web Tokens)
* **Database:** SQLite (Development) / PostgreSQL (Production ready)
* **Code Execution:** Secure sub-process execution for Python/Java/C++
